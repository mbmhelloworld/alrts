# alrts

alrts is a example webapp for users finding Live Events and getting alerts for them.

## scripts
1. Run MongoDB: 
`mongod`
2. frontend
    - start
    - build
    - test
    - eject
    - lint
3. backend
    - start
    - dev
    - watch
    - build

## Docker
`docker-compose up`
[localhost:3000](http://localhost:3000)
